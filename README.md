# XCELERATED Next.js Website

Multi-page Next.js starter for XCELERATED with:
- Home, About, Programs, XSPA, XPT, XGX, Register, Contact
- black / gold / white / gray design
- hero video section
- dropdown / mega menu behavior
- responsive layout for desktop, tablet, and mobile
- custom SVG logo and favicon
- placeholder media files you can replace

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm install
npm run build
npm run start
```

## Replace media
- Logo: `public/logo/xcelerated-logo.svg`
- Favicon: `public/favicon.svg`
- Hero video: `public/video/hero.mp4`
- Images: `public/images/*.svg`

## Notes
- Contact details are placeholders and should be replaced.
- Forms are frontend only in this starter.
- XGX pricing is left flexible because no final amount was provided.
